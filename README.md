# angit

Springboot + Mybatis + redis + angularjs

下载
git clone https://github.com/ruyangit/angit.git

运行
cd angit
mvn install
$cd angit-intra/target 
$java -jar angit-intra.jar &

运行失败，请倒入eclipse 开发工具查看项目结构
